Muscari Server Since 2017 Nov.


Muscari Serverが選ばれる理由

    1.建築、ワールド観
  • 一般的なMODサーバーにありがちな
    ワールド美化への無配慮は非推奨。
    (スカイブロック時を除く)
    シーズンによって地形生成modや
    ブロック変形modなどが扱われている。
    通常のMinecraftで成し得なかった本当の美を
    modの力で昇華させよう。

    2.世界中で人気なmod達が目の前に
  • 国内のマイクラ鯖の中でもModサーバーは
    絶滅危惧種。
    Curseforgeから選りすぐりのmodを
    開発陣が選出。
    1000万ダウンロード以上の数々のmodに
    触れてみよう。

    3.独自ランチャーによる簡単参加
  • 独自ランチャー「FruitLauncher」
    によりmod導入の弊害を排除。
    誰でも簡単にMuscari Server
    での生活を始められるように。

    4.個性的な仲間たち
  • マルチサーバーの特権、
    他プレイヤーとの交流が楽しめる。
    modを遊ぶ上での不安も、
    そんな心配はここでは無用。
    mod初心者にもおすすめのmodサーバー。
    分からないことがあったら数々の
    modのエキスパートに聞いてみよう。
